HTML

Es un lenguaje de marcado de Hipertexto, que se utilila para definir la estructura de una pagina web, es decir, que con este lenguaje creamos el esqueleto de la pagina. Utiliza etiqueta para definir la funcion que cumple cada elemento dentro de la pagina, a la vez estas poseen atributos que permiten especificar alguna caracteristica de dichos elementos. Las etiquetas HTML se clasifican de diferentes maneras. Por su tipo, en Sematicas y Presentacionales, o por su representacion dentro de la pagina, como se muestra a continuacion.

  • DE BLOQUE: son aquellos elementos que ocupan todo el ancho de su contenedor, se les puede definir tamaño, generan salto de linea y generalmente funcionan como contenedores
  • DE LINEA: son aquellos elementos que ocupan el ancho de su contenido, su tamaño es automatico, comparten espacio con otras etiquetas de su tipo y generalmente funcionan como contenido

CSS

Las hojas de estilo en cascada corresponden a un lenguaje de Diseño Grafico, que utiliza selectores y reglas para definir el aspecto de los elementos que se representan en la pagina web. En este caso el orden de los factores NO altera el producto.
Exiten diferentes formas de implementar CSS en un sitio web y a continuacion se detallan las caracteristicas de las mismas

  • EN LINEA: el codigo se escribe dentro de la etiqueta utilizando el atributo style="...". Es el estilo mas especifico y por lo tanto, con mayor prioridad, pero es bastante impractico.
  • EN BLOQUE: el codigo se escribe separado de la estructura y se utilizan selectores para aclarar a quien le corresponde dicho estilo. Se puede utilizar dentro de la pagina, como en un archivo aparte.
